# Productivity Dashboard

A modern productivity dashboard built with HTML, CSS, and JavaScript.

## Features

- Live Weather & Clock
- To-Do List
- Daily Planner
- Motivation Quotes
- Pomodoro Timer
- Dynamic Day/Night Backgrounds
- Local Storage Support
